# Archive — Original Drafts

This folder preserves the original, collaborative source material for the GrayArea project.

- Original source document: `_LATEST-2023-12-26 NZ w-Danny-RY.docx` — the working manuscript co-authored by Rene and Daneel (and referenced collaborators). This `.docx` is the authoritative archival copy.
- Markdown conversion: the `.docx` was converted to Markdown and saved under `archive/original/md/` for easier review and edit-tracking. Files there are "cut-off" exports (suitable for editing and version control) and are preserved for editorial history.

Purpose

- Preservation: keep the original collaborative drafts intact for provenance, attribution, and editorial history.
- Auditability: retain the converted Markdown as a readable, diffable snapshot of the original manuscript.
- Workflow: do not edit files in `archive/original/md/` directly for canonical Book 1 content. To propose changes, copy relevant sections into `book1/chapters/` or open a PR with suggested edits. This keeps the archive authoritative and the working draft flow clean.

Notes

- Timeframe: these drafts were authored ~3–5 years before the current repository snapshot (see file timestamps and `ORIGINAL.md` for more context).
- If you need the full commit history associated with these drafts, consider extracting `archive/original` into a separate repository (see `git subtree split` or `git filter-repo`).

Contact
